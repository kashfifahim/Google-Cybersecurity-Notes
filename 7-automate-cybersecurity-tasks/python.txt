Specific areas of cybersecurity in which Python might be used to automate specific tasks:

- malware analysis
- access control list management 
- intrusion detection 
- compliance checks 
- network scanning 

a notebook is an online interface for writing, storing, and running code 
code cells are meant for writing and running code 
a notebook provides a machanism for running these code cells 
markdown cells are meant for describing the code
